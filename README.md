## Özellikler

- Todo ekleme, silme ve tamamlandı olarak işaretleme ✅  
- Filtreleme: **All / Active / Completed**  
- LocalStorage ile tarayıcıda veri saklama  
- Modern, yuvarlak ve eğlenceli UI (Tailwind)  
- Responsive ve hover efektleri
